# C++ 默认函数介绍

## 1. 构造函数
```cpp
class A {
public:
    A();
    ~A();
};
```

## 2. 析构函数
```cpp
class A {
public:
    ~A();
};
```

## 3. 拷贝构造函数
```cpp
class A {
public:
    A(const A& other);
};
```

## 4. 赋值运算符重载
```cpp
class A {
public:
    A& operator=(const A& other);
};
```

## 5. 移动构造函数
```cpp
class A {
public:
    A(A&& other);
};
```

## 6. 移动赋值运算符重载
```cpp
class A {
public:
    A& operator=(A&& other);
};
```
