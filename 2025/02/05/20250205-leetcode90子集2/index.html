题目

90. 子集 II

给你一个整数数组 nums，其中可能包含重复元素，请你返回该数组所有可能的子集（幂集）。
解集不能包含重复的子集。返回的解集中，子集可以按任意顺序排列。

解题思路

1. 回溯

原理

1. 排序的作用

  • 将数组排序，使相同元素相邻
  • 便于后续去重处理
  • 例如：[4,1,4,4,4] → [1,4,4,4,4]

2. 回溯过程

  • 每个元素都有”选”和”不选”两种状态
  • 通过 start 参数控制选择的起始位置
  • 使用 path 记录当前路径

3. 去重机制

  • 对于重复元素，只在第一次遇到时使用
  • i > start and nums[i] == nums[i-1] 跳过同层重复元素
  • 例如：对于 [1,2,2]

4. 复杂度分析

  • 时间复杂度：O(n×2^n)，n是数组长度
  • 空间复杂度：O(n)，递归栈的深度

5. 关键点

  1. 必须先排序，这是去重的基础
  2. 使用 path[:] 创建路径副本
  3. 去重条件 i > start 不能省略
  4. 递归时使用 i+1 而不是 start+1
